## 1. 프로젝트 개요
본 프로젝트는 **온도 및 조도 센서를 기반으로 LED와 서보모터를 자동 제어**하는 지능형 임베디드 시스템을 구현하는 것을 목표로 한다.  
Arduino 기반의 센서 네트워크를 통해 실시간 환경 데이터를 수집하고, Bluetooth 통신으로 모바일 앱과 연동하여 **스마트한 생활 공간 관리**를 가능하게 한다.

---

## 2. 핵심 아이디어
- 실내 조명 및 환경 제어를 자동화하여 에너지 절약과 편리성을 동시에 제공  
- Arduino + HC-06 블루투스 모듈을 활용해 실시간 모니터링 및 원격 제어 가능  
- 모바일 앱을 통해 환경 상태를 확인하고 수동 제어 지원

---

## 3. 주요 기능
| 기능 | 설명 |
|------|------|
| **실시간 센서 모니터링** | 온도, 조도 센서 데이터를 수집 및 표시 |
| **자동 제어 시스템** | 온도/조도 조건에 따라 LED 및 서보모터 자동 제어 |
| **무선 통신 기능** | 블루투스(HC-06) 기반으로 모바일 기기와 실시간 연동 |
| **모바일 앱 제어** | Android 앱을 통한 실시간 제어 및 상태 확인 기능 |

---

## 4. 시스템 구성도
```plaintext
[Temperature Sensor]──┐
[Light Sensor]─────────┼──> Arduino Uno ──> Servo Motor
                        │
                        └──> LED ───> Bluetooth(HC-06) ──> Mobile App
```

## 5. 주요 부품 목록

## 7. 문제 정의 및 해결책
**문제점:**  
- 기존의 조명 및 환경 제어 시스템은 사용자가 수동으로 조작해야 하며, 실시간 환경 변화에 대응하기 어려움  
- 외부에서도 시스템을 제어할 수 없음  

**해결책:**  
- 센서 기반 임베디드 시스템으로 자동화 구현  
- 블루투스 통신을 통해 모바일 기기에서 원격 제어 가능  
- 실시간 모니터링을 통해 효율적인 에너지 사용 및 사용자 편의성 향상
